# twiggit

Pragmatic git worktree management tool with focus on rebase workflows.

## Installation

```bash
curl -fsSL https://gitlab.com/amoconst/twiggit/-/raw/main/install.sh | bash
```

Or download manually from: https://gitlab.com/amoconst/twiggit/-/releases

## Quick Start

```bash
# Show version
twiggit version

# List worktrees
twiggit list

# Create a worktree
twiggit create myfeature
```
